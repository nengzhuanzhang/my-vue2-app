# Vue2-vant2

## 项目简介

- Vue2 + vant2 H5 商城

> 项目环境

- vue@2
- vue-cli@4
- vue-router@3
- vuex@3
- vant@2
- axios
- sass

## 使用

```bash
# 安裝
npm install
# 运行
npm run serve
# 单元测试
npm run test:unit
# 打包
npm run build
```
